  • Text Align

    使用 .b-text-left_ .b-text-center_ .b-text-right_ 設定文字對齊。

    text-left text-center text-right
    .b-text-left_ .b-text-center_ .b-text-right_
  • Border

    使用 .b-solid_ .b-dashed_ 設定邊框樣式。

    .b-solid_ .b-dashed_
    <div class="b-solid_ is-all"></div>
    <div class="b-dashed_ is-all"></div>
    <div class="b-solid_ is-TB"></div>
    <div class="b-dashed_ is-TB"></div>
    <div class="b-solid_ is-RL"></div>
    <div class="b-dashed_ is-RL"></div>
    <div class="b-solid_ is-T"></div>
    <div class="b-dashed_ is-T"></div>
    <div class="b-solid_ is-B"></div>
    <div class="b-dashed_ is-B"></div>
    <div class="b-solid_ is-R"></div>
    <div class="b-dashed_ is-R"></div>
    <div class="b-solid_ is-L"></div>
    <div class="b-dashed_ is-L"></div>
  • Display

    使用 .b-hide_ .b-block_ .b-inline_ 設定 display。

    .b-hide_ .b-block_ .b-inline_
    在任何 tag 使用 .b-hide_ 可以讓任何的 tag 改變 display 為 none 在任何 tag 使用 .b-block_ 可以讓任何的 tag 改變 display 為 block 在任何 tag 使用 .b-inline_ 可以讓任何的 tag 改變 display 為 inline-block
  • Paragraph

    使用 .b-p_ 預設段落字級為 14px，可使用 .is-lg 改變預設字級為 15px ，.is-sm 改變預設字級為 12px。
